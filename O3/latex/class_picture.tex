De klasse {\ttfamily \mbox{\hyperlink{class_picture}{Picture}}} is een subklasse van de klasse {\ttfamily \mbox{\hyperlink{class_object}{Object}}} en stelt een afbeelding voor die kan worden getekend op een {\ttfamily sf\+::\+Render\+Window} venster. De klasse bevat een pad naar een afbeeldingsbestand, een {\ttfamily sf\+::\+Texture} object en een {\ttfamily sf\+::\+Sprite} object. Het heeft een constructor die een positie en een pad naar een afbeeldingsbestand verwacht. Ook heeft het een {\ttfamily draw} functie die de afbeelding tekent op een gegeven venster, een {\ttfamily get\+Bounding\+Box} functie die de begrenzende rechthoek van de afbeelding retourneert, een {\ttfamily get\+Position} functie die de positie van de afbeelding retourneert en een {\ttfamily update\+Position} functie die de positie van de afbeelding bijwerkt.
