Detailed Description

De klasse ball is verantwoordelijk voor het maken van een cirkelvormig object.

Deze klasse is een subklasse van de klasse Object. De klasse heeft een cirkelvormig object met een bepaalde grootte en kleur. Het heeft een constructor die een positie, grootte en kleur van de cirkel als invoer neemt. Ook heeft het een functie om de cirkel te tekenen op een venster, een functie om de omtrek van de cirkel te krijgen, een functie om de positie van de cirkel te krijgen, en een functie om de positie van de cirkel bij te werken.

Note
De positie is de positie van de linkerbovenhoek van de cirkel.
